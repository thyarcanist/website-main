Carrier | Circuit Board Commissioning

Second Carrier project; this one required creating an [0][0][0] LED System that would communicate with the Circuit Board.
The user would press buttons to cycle through the menu to check the installation, diagnose and troubleshoot.
This is a (30+) two-part course where the user gains the knowledge of troubleshooting via the mobile app and circuit board.
Playable via Tablet, PC, and VR. Created the core simulation gameplay for the LED System and finished the co-workers mobile phone app.

Technologies used: Unity, C#, VR
